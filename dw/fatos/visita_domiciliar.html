    Tabela 1. Apresentação das colunas x campos múltipla escolha

    Além das colunas apresentadas na Tabela 1, algumas foram criadas para facilitar o controle sobre os dados registrados:

    • st_mot_vis_busca_ativa: responsável por identificar se alguma opção do grupo Busca ativa foi selecionada;
    • st_mot_vis_acompanhamento: responsável por identificar se alguma opção do grupo Acompanhamento foi selecionada;
    • st_mot_vis_ctrl_ambnte_vetor: até a versão 2.0, esta opção era única. A partir da versão 2.1, esta coluna foi transformada em um agrupador, responsável por identificar se alguma opção do grupo Controle ambiental / vetorial foi selecionada;
    • co_uuid_origem_fcd: responsável por armazenar a informação de qual domicílio recebeu a visita. Corresponde ao campo nu_uuid_ficha_origem da tabela tb_fat_cad_domiciliar.
